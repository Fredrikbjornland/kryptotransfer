## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`cd client %% npm install`  

To Start Server:

Add .env with giphy token

`npm run dev`  

To Visit App:

`localhost:3000`
